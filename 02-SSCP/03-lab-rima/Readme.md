**Lab Setup**

- Ubuntu Linux VM – server to scan and monitor
- Windows Server VM – AD, file server
- Tools: `Nmap`, `OpenVAS`, `Nessus`, `fail2ban`, `auditd`, `Splunk` or `ELK Stack` (optional)

**Real-Time Examples**

| Concept         | Example                                                                                   |
| --------------- | ----------------------------------------------------------------------------------------- |
| Asset           | Customer database, web server, application server                                         |
| Threat          | Malware, phishing, insider sabotage                                                       |
| Vulnerability   | Outdated software, weak password, misconfigured firewall                                  |
| Risk Analysis   | Web server exposed to the internet → high risk for attack                                 |
| Risk Response   | Patch server (mitigation), purchase cyber insurance (transfer), remove server (avoidance) |
| Risk Monitoring | Daily log review, vulnerability scans, SOC alerts                                         |
