**Lab Setup**

- Ubuntu Linux VM – encryption tasks
- Windows Server/Client VM – PKI and certificate tasks
- `Tools:` `OpenSSL`, `GPG`, `LUKS`, `ssh-keygen`, `openssl req`

**Real-Time Examples**

| Concept               | Example                                            |
| --------------------- | -------------------------------------------------- |
| Symmetric Encryption  | AES encrypting sensitive database backups          |
| Asymmetric Encryption | RSA used to exchange session keys in TLS handshake |
| Hashing & Salting     | Storing user passwords securely in a database      |
| Digital Signature     | Signing software updates to verify authenticity    |
| PKI                   | Web server certificate issued by a trusted CA      |
| Data in transit       | HTTPS between client browser and web server        |
| Data at rest          | BitLocker encrypting laptop drive                  |
