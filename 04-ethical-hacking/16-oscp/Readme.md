## Offensive Security Certified Professional - OSCP

### OSCP

OSCP is a certification offered by Offensive Security that proves you can:

- Think like a hacker
- Exploit real systems
- Perform penetration testing in a controlled environment

> 👉 It is not theory-based — it is 100% practical.

### Socket programming

It's a technique used to establish communication between two devices over a network using sockets.

- A socket acts as an endpoint defined by an IP address and port number.
- It allows sending and receiving data using protocols like TCP or UDP.
