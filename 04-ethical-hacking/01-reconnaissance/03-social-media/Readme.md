## Overview

Both Followerwonk and Sherlock are popular tools used in reconnaissance (OSINT), but they serve different purposes.

### Comparison

| Feature        | Followerwonk          | Sherlock               |
| -------------- | --------------------- | ---------------------- |
| Platform Focus | Twitter only          | 300+ websites          |
| Type           | Web tool              | CLI (Python tool)      |
| Use Case       | Social media analysis | Username investigation |
| Skill Level    | Beginner              | Intermediate           |
